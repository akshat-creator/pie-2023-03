ABOUT OUR PROJECT

Over the course of this project, we aimed to create a fun and interactive Skee Ball game that users can play. Our main goals for every component of this project were to prioritize creating a clean and polished display we would be proud of. The overall mechanical goals for this project consisted of creating a solid structure for repeated gameplay. On the electrical side, we wanted to focus on a clean wiring harness and a power detection module. From the software standpoint, our main priority was creating a polished score display.
